# Clone Básico do Spotify (HTML + CSS + JavaScript)

Um clone simples e funcional de uma interface inical do Spotify, feito apenas com **HTML, CSS e JavaScript puro** (sem frameworks). 


<a href="https://ileonardo-lima.github.io/Clone_Basico_Spotify/">Clique para acessar!</a>

## Funcionalidades implementadas

- Layout responsivo que imita o Spotify Web Player (2024/2025)
- Sidebar esquerda com menu de navegação
- Header com busca e perfil do usuário
- Home com cards de playlists, álbuns e artistas
- Player inferior fixo com controles de reprodução (play/pause, próximo/anterior, volume, progresso)
- Reprodução real de músicas de exemplo (áudio local ou via link público)
- Efeitos hover, scroll personalizado e transições suaves
- Tema escuro nativo (igual ao Spotify)

## Tecnologias usadas

- HTML5
- CSS3 
- JavaScript
